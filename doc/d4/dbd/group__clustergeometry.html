HEADER_PREFIX pclustergeometry build_from_objects_clustergeometry ( pobjects  objs)

Construct a cluster geometry from objects.

Constructs an d-dimensional cluster geometry object, with d being the dimension of the space containing the objects, and sets the characteristic points and coordinates for the support bounding of each indice to the coordinate of corresponding object.

Parameters
objsObjects from which the cluster geometry will be constructed.
Returns
The constructed cluster geometry object.
