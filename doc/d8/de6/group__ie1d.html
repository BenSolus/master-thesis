Detailed Description

Approximation of one-dimensional integral equation.

The ie1d class is used to approximate the simple one-dimensional Fredholm integral equation

\[ -\int_0^1 \ln |x-y| u(y) \,dy = f(x) \qquad\text{ for all } x\in[0,1] .\]
